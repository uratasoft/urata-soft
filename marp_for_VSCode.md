# ChatGPTとは

- 自然言語処理タスクに強力な性能を発揮
- テキスト生成や質問応答などのタスクが得意

# Marp for VS Codeとは

- Visual Studio Codeで使えるMarkdownベースのスライド作成ツール
- シンプルで使いやすい
- コードハイライトや画像の埋め込みが可能

# ChatGPTとMarp for VS Codeを組み合わせる方法

3. スライドを完成させる
